📖 Hướng Dẫn Sử Dụng

🔑 1. Tạo GitHub Personal Access Token

  1. Vào Settings → Developer settings → Personal access tokens
  2. Chọn Tokens (classic)
  3. Click Generate new token
  4. Chọn quyền: repo (full control of private repositories)
  5. Copy token và dán vào ô "GitHub Personal Access Token"

📝 2. Viết Bài Mới

  • Nhập Tiêu Đề bài viết
  • Chọn Categories (ví dụ: Security, CVE)
  • Thêm Tags (ví dụ: sqli, rce, writeup)
  • Viết nội dung bằng Markdown
  • Click 🚀 Publish để công bố

📚 3. Quản Lý Bài Viết

  • Vào tab 📚 Danh Sách Bài
  • Click bài để chỉnh sửa
  • Click nút 🗑️ Delete để xóa
