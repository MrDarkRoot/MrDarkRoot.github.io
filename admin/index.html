📖 Hướng Dẫn Tạo Token

  1. Vào https://github.com/settings/tokens
  2. Chọn Generate new token (classic)
  3. Tên: blog-admin
  4. Scope: Chọn repo (full control of private repositories)
  5. Copy token và dán vào ô trên
  6. Click 💾 Save Token
